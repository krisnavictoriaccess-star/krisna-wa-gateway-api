Dokumentasi Teknis API Lengkap (Base URL, Parameter, dan Respon)

🌍 Base URL (Akar URL)

Semua endpoint di bawah ini harus dipanggil dengan awalan URL berikut:

https://api.krisnamarket.my.id

🔐 Autentikasi (Headers)

Sistem ini menggunakan autentikasi berbasis Header. Anda wajib menyertakan header berikut pada setiap request sesuai dengan tingkat akses endpoint.

Nama Header Tipe Status Deskripsi
x-master-key String Wajib (Admin) Kunci rahasia master dari VPS. Hanya digunakan untuk endpoint bertanda Master.
x-api-key String Wajib (Klien) Token API unik milik klien. Digunakan untuk endpoint bertanda User.
Content-Type String Wajib Harus selalu bernilai application/json untuk request bertipe POST.
sender_id String Opsional Khusus endpoint pengiriman pesan. Jika diisi nomor (ex: 62812xxx), pesan akan dikirim dari nomor tersebut. Jika kosong, sistem menggunakan Rotator Otomatis.

Body Parameters (JSON)

FieldTypeStatusDeskripsi

labelStringOpsionalNama/label klien. Default: User.
expiry_daysIntegerOpsionalMasa aktif kustom dalam hari (ex: 365). Jika kosong, mengikuti bawaan paket.

  "label": "Klien VIP A",
  "expiry_days": 30
}
Respon Sukses (201 Created) 
{

  "message": "API Key Premium berhasil dibuat.",
  "plain_key": "KEY-ABC123DEF456",

{
  "status": false,
  "message": "Paket tidak valid."
}

⚡ Real-Time Events (WebSocket)

SOCKET.IO WSS Connection Dengarkan perubahan status WhatsApp secara instan (QR Code / Pairing Code)

Gateway kami juga memancarkan sinyal langsung (real-time) setiap kali ada perubahan status pada nomor WhatsApp klien (misalnya saat meminta QR Code, Pairing Code, atau saat koneksi terputus). Anda dapat mendengarkan (*listen*) *event* ini di *frontend* (Web/Mobile) Anda menggunakan *library* Socket.IO Client.

Cara Koneksi di Frontend (JavaScript)

<script src="https://cdn.socket.io/4.7.2/socket.io.min.js"></script>
<script>
  // 1. Hubungkan ke Server Gateway
  const socket = io("https://api.krisnamarket.my.id");

  // 2. Dengarkan event 'device_status'
  socket.on("device_status", (data) => {
      console.log("Status Baru:", data);
      
      if(data.status === "WAITING_PAIRING") {
          alert("Kode Pairing Anda: " + data.code);
      } else if (data.status === "WAITING_QR") {
          // data.qr berisi string Base64 gambar QR Code
          document.getElementById('qr-image').src = data.qr;
      } else if (data.status === "CONNECTED") {
          alert("WhatsApp " + data.device + " Berhasil Terhubung!");
      } else if (data.status === "DISCONNECTED") {
          alert("WhatsApp " + data.device + " Terputus dari server.");
      }
  });
</script>

Tabel Payload (Struktur Data device_status)

StatusDeskripsiProperty Tambahan
WAITING_PAIRINGMenunggu Anda memasukkan kode di HP.code: Berisi 8 digit kode angka/huruf.
WAITING_QRMenunggu Anda melakukan scan barcode.qr: Berisi URL Base64 Gambar QR Code.
CONNECTEDNomor WhatsApp berhasil tersambung & siap dipakai.-
DISCONNECTEDNomor WhatsApp terputus (Keluar/Log Out).-
