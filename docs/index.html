Dokumentasi Teknis API Lengkap (Base URL, Parameter, dan Respon)

🌍 Base URL (Akar URL)

Semua endpoint di bawah ini harus dipanggil dengan awalan URL berikut:

https://api.krisnamarket.my.id

🔐 Autentikasi (Headers)

Sistem ini menggunakan autentikasi berbasis Header. Anda wajib menyertakan header berikut pada setiap request.

Nama HeaderTipeStatusDeskripsi
x-master-keyStringWajib (Admin)Kunci rahasia master dari VPS.
x-api-keyStringWajib (Klien)Token API unik milik klien.
Content-TypeStringWajibSelalu bernilai application/json (POST).
sender_idStringOpsionalDigunakan pada Messaging untuk menentukan pengirim spesifik.

👑 Admin (Master Key)

GET /api-key/list Melihat semua klien terdaftar
Requires x-master-key
Contoh Respon Sukses (200 OK) 
{
  "status": true,
  "data": [
    {
      "key": "KEY-XXXX",
      "label": "User A",
      "paket": "Premium"
    }
  ]
}
Contoh Respon Error (401 Unauthorized) 
{
  "status": false,
  "message": "Akses ditolak. Master Key salah."
}
POST /api-key/generate Membuat API Key baru untuk klien
Requires x-master-key

Body Parameters (JSON)

FieldTypeStatusDeskripsi
paketStringOpsionalPilihan: Free, Lite, Pro, Premium. Default: Free.
labelStringOpsionalNama/label klien. Default: User.
expiry_daysIntegerOpsionalMasa aktif kustom dalam hari (ex: 365).

Contoh Request Body

{
  "paket": "Premium",
  "label": "Klien VIP A",
  "expiry_days": 30
}
Respon Sukses (201 Created) 
{
  "status": true,
  "plain_key": "KEY-ABC123DEF456"
}
Respon Error (400 Bad Request) 
{
  "status": false,
  "message": "Paket tidak valid."
}
POST /api-key/extend Menambah masa aktif klien
Requires x-master-key

Contoh Request Body

{
  "target_api_key": "KEY-XXXXX",
  "tambah_hari": 30
}
POST /api-key/delete Menghapus total API Key klien
Requires x-master-key

Contoh Request Body

{
  "target_api_key": "KEY-XXXXX"
}

✉️ Messaging (Pengiriman Pesan)

POST /kirim-pesan Kirim Pesan Teks (Mendukung Rotator)
Requires x-api-key

Body Parameters (JSON)

FieldTypeStatusDeskripsi
nomorStringWajibNomor tujuan (Contoh: 628123456789).
pesanStringWajibIsi teks pesan WhatsApp.

Contoh Request Body

{
  "nomor": "628123456789",
  "pesan": "Halo, ini pesan dari *API Gateway*! \n\nBaris baru didukung."
}
Respon Sukses (200 OK) 
{
  "status": true,
  "message": "Pesan dimasukkan ke antrean pengiriman.",
  "queue_id": 1542
}
Respon Error (402 Payment Required) 
{
  "status": false,
  "message": "Kuota pesan Anda habis. Silakan upgrade."
}
POST /kirim-massal Kirim Pesan Broadcast (Spintax & Delay)
Requires x-api-key

Contoh Request Body

{
  "delay_ms": 3000,
  "data": [
    { "nomor": "6281xxx", "pesan": "Halo bos" },
    { "nomor": "6282xxx", "pesan": "Halo gan" }
  ]
}
POST /kirim-media Kirim Gambar / Video / Dokumen
Requires x-api-key

Contoh Request Body

{
  "nomor": "628123456789",
  "media_url": "https://domain.com/gambar.jpg",
  "media_type": "image",
  "caption": "Silakan lihat brosur terlampir."
}
POST /kirim-polling Kirim Polling (Voting) WhatsApp
Requires x-api-key

Contoh Request Body

{
  "nomor": "628123456789",
  "name": "Berapa umur Anda?",
  "values": ["18-25", "26-35"],
  "selectableCount": 1
}
POST /kirim-grup Kirim Pesan ke Grup WhatsApp
Requires x-api-key

Contoh Request Body

{
  "group_id": "123456789@g.us",
  "pesan": "Halo member grup!"
}

📱 Device Management (Sesi WA)

POST /device/add Membuat sesi WhatsApp baru (Pairing Code)
Requires x-api-key

Contoh Request Body

{
  "nomor_hp": "6285551234"
}
Respon Sukses (201 Created) 
{
  "status": true,
  "message": "Sesi berhasil diinisiasi.",
  "pairing_code": "A1B2-C3D4"
}
POST /device/delete Logout dan hapus sesi WhatsApp secara paksa
Requires x-api-key

Contoh Request Body

{
  "nomor_device": "6285551234"
}

🤖 Webhooks & Chatbot (Auto-Reply)

POST /webhook/set Mendaftarkan URL Webhook untuk menerima Event
Requires x-api-key

Contoh Request Body

{
  "webhook_url": "https://serveranda.com/api/wa-callback"
}
POST /auto-reply/add Menambahkan aturan balasan otomatis (Chatbot)
Requires x-api-key

Contoh Request Body

{
  "nomor_device": "628123456789",
  "keyword": "harga",
  "response": "Harga paket kami mulai dari Rp 50.000,-",
  "match_type": "contains"
}

📊 Data & Antrean

Respon Sukses (200 OK)

  "data": {
    "paket": "Pro",
    "limit_pesan": 50000,
    "sisa_kuota": 48480
  }
