Krisna WA Gateway

Dokumentasi Resmi Integrasi API Enterprise

🌍 Base URL

https://api.krisnamarket.my.id

🔐 Autentikasi (Headers)

Sertakan kredensial di header HTTP pada setiap request.

HeaderStatusDeskripsi
x-master-keyWajib (Admin)Untuk akses endpoint master / admin (manajemen klien).
x-api-keyWajib (Klien)Token klien yang didapatkan dari Generate API Key.
sender_idOpsionalDigunakan pada fitur Messaging untuk menetapkan device/nomor pengirim spesifik (format: 628xxx).

📱 Device Management (Sesi WA)

POST /device/add Menyambungkan nomor WhatsApp ke server untuk mendapatkan Pairing Code (Kode Tautan).
Requires x-api-key

Body JSON

FieldTypeStatusDeskripsi
nomor_hpStringWajibNomor HP WA yang akan dihubungkan (format: 628xxx).

Contoh Request Body

{ "nomor_hp": "6285551234" }
Respon Sukses (200 OK) 
{
  "status": true,
  "message": "Silakan masukkan kode pairing ini di WhatsApp.",
  "pairing_code": "V5Y2A9XR",
  "device_status": "WAITING_PAIRING"
}

Catatan: Anda harus membuka aplikasi WhatsApp di HP Anda, pilih menu Tautkan Perangkat > Tautkan dengan Nomor Telepon Saja, lalu masukkan kode 8 karakter dari respon di atas.

GET /device/list Melihat daftar device milik Anda yang saat ini tersambung ke server.
Requires x-api-key
POST /device/settings Mengubah pengaturan otomatisasi pada perangkat spesifik.
Requires x-api-key

Body JSON

FieldTypeStatusDeskripsi
nomor_deviceStringWajibNomor device Anda (628xxx).
reject_callBooleanOpsionalOtomatis menolak telepon masuk (true/false).
autoreadBooleanOpsionalOtomatis membaca (centang biru) pesan masuk (true/false).
record_inboxBooleanOpsionalMenyimpan riwayat chat masuk ke database (true/false).
POST /device/delete Logout dan putuskan sambungan WhatsApp Anda dari server.
Requires x-api-key 
{ "nomor_device": "6285551234" }

⚡ Real-Time Events (WebSocket)

SOCKET.IO WSS Connection Dengarkan pembaruan status Pairing Code secara instan.

Saat Anda memanggil /device/add, server akan meminta WhatsApp untuk menghasilkan Pairing Code (8 digit). Agar antarmuka Anda bisa merespon secara langsung (Real-Time) tanpa perlu memuat ulang (*refresh*), Anda wajib menggunakan Socket.IO Client.

Cara Koneksi di Frontend (JavaScript)

<script src="https://cdn.socket.io/4.7.2/socket.io.min.js"></script>
<script>
  const socket = io("https://api.krisnamarket.my.id");

  socket.on("device_status", (data) => {
      console.log("Pembaruan Status:", data);
      
      if(data.status === "WAITING_PAIRING") {
          alert("KODE PAIRING ANDA: " + data.code);
          // Tampilkan data.code ini ke layar besar Anda agar bisa dibaca jelas
      } else if (data.status === "CONNECTED") {
          alert("WhatsApp Berhasil Terhubung!");
      } else if (data.status === "DISCONNECTED") {
          alert("WhatsApp Terputus / Logout.");
      }
  });
</script>

Tabel Payload device_status

StatusDeskripsiProperty Tambahan
WAITING_PAIRINGServer telah mengirimkan kode rahasia.code: Berisi 8 digit kode kombinasi.
CONNECTEDWhatsApp berhasil diotorisasi.-
DISCONNECTEDSesi terputus (Keluar/Dihapus paksa).-
