WhatsApp Gateway Enterprise

Dokumentasi Resmi API - Anti Blokir & Ringan

🔐 Autentikasi (Headers)

Hampir semua endpoint membutuhkan header autentikasi berikut:

  • x-master-key : Digunakan khusus untuk endpoint Admin.
  • x-api-key : Digunakan oleh klien/pengguna biasa.

👑 Admin (Master Key)

GET /api-key/list Melihat semua klien terdaftar
Requires x-master-key

Responses

{
  "status": true,
  "data": [ ... ]
}
POST /api-key/generate Membuat API Key baru untuk klien
Requires x-master-key

JSON Body

{
  "paket": "Premium",
  "label": "Klien VIP",
  "expiry_days": 30
}
POST /api-key/extend Menambah masa aktif hari klien
Requires x-master-key

JSON Body

{
  "target_api_key": "KEY-XXXXX",
  "tambah_hari": 30
}
POST /api-key/delete Menghapus total API Key beserta semua sesi perangkat
Requires x-master-key

JSON Body

{
  "target_api_key": "KEY-XXXXX"
}

📱 Device Management

POST /device/add Membuat sesi WhatsApp baru (Pairing Code)
Requires x-api-key

JSON Body

{
  "nomor_hp": "62812xxxx"
}
POST /device/delete Logout dan hapus sesi WhatsApp
Requires x-api-key

JSON Body

{
  "nomor_device": "62812xxxx"
}

✉️ Messaging (Kirim Pesan)

POST /kirim-pesan Kirim Pesan Teks (Mendukung Rotator)
Requires x-api-key

Gunakan header opsional sender_id jika ingin memilih pengirim spesifik. Kosongkan untuk Rotator otomatis.

JSON Body

{
  "nomor": "62812xxxx",
  "pesan": "Halo ini pesan dari API!"
}
POST /kirim-massal Kirim Pesan Broadcast (Spintax & Delay)
Requires x-api-key

JSON Body

{
  "delay_ms": 3000,
  "data": [
    { "nomor": "6281xxx", "pesan": "Halo bos" },
    { "nomor": "6282xxx", "pesan": "Halo gan" }
  ]
}
POST /kirim-media Kirim Gambar / Video / Dokumen
Requires x-api-key

JSON Body

{
  "nomor": "62812xxxx",
  "media_url": "https://example.com/image.jpg",
  "media_type": "image",
  "caption": "Keterangan gambar"
}
POST /kirim-polling Kirim Polling (Voting) WhatsApp
Requires x-api-key

JSON Body

{
  "nomor": "62812xxxx",
  "name": "Berapa umur Anda?",
  "values": ["18-25", "26-35"],
  "selectableCount": 1
}
POST /kirim-grup Kirim Pesan ke Grup WhatsApp
Requires x-api-key

JSON Body

{
  "group_id": "123456789@g.us",
  "pesan": "Halo member grup!"
}
