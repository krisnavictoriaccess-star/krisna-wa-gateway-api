Selamat Datang di API Gateway

Dokumentasi resmi untuk integrasi sistem Anda dengan Enterprise WhatsApp Gateway.

🌍 Base URL (Akar URL)

Semua endpoint yang ada di menu navigasi harus digabung (disambung) dengan awalan URL berikut:

https://api.krisnamarket.my.id

🔐 Konfigurasi Autentikasi (Headers)

Sistem ini menggunakan autentikasi berbasis Header. Anda wajib menyertakan header berikut pada setiap permintaan (request).

Nama HeaderTipeStatusDeskripsi
x-master-keyStringWajib (Admin)Kunci rahasia master dari VPS Anda. Digunakan khusus untuk operasi sensitif.
x-api-keyStringWajib (Klien)Token API unik milik klien (yang dibuat melalui Generate API Key).
Content-TypeStringWajibSelalu bernilai application/json untuk semua tipe pengiriman POST.
sender_idStringOpsionalDigunakan pada fitur Messaging untuk memilih nomor pengirim spesifik (Jika Anda memiliki banyak device). Kosongkan jika ingin mode Rotator Otomatis.

List API Key

Melihat semua akun klien yang terdaftar di dalam database.

GET /api-key/list
Requires x-master-key
Contoh Respon Sukses (200 OK) 
{
  "status": true,
  "data": [
    {
      "key": "KEY-XXXX",
      "label": "User A",
      "paket": "Premium"
    }
  ]
}
Contoh Respon Error (401 Unauthorized) 
{
  "status": false,
  "message": "Akses ditolak. Master Key salah."
}

Generate API Key

Membuat kredensial klien baru agar mereka bisa menggunakan WhatsApp Gateway Anda.

POST /api-key/generate
Requires x-master-key

Body Parameters (JSON)

FieldTypeStatusDeskripsi
paketStringOpsionalPilihan: Free, Lite, Pro, Premium. Default: Free.
labelStringOpsionalNama/label klien. Default: User.
expiry_daysIntegerOpsionalMasa aktif kustom dalam hari (ex: 365).

Contoh Request Body

{
  "paket": "Premium",
  "label": "Klien VIP A",
  "expiry_days": 30
}
Respon Sukses (201 Created) 
{
  "status": true,
  "plain_key": "KEY-ABC123DEF456"
}
Respon Error (400 Bad Request) 
{
  "status": false,
  "message": "Paket tidak valid."
}

Extend API Key

Menambahkan masa aktif (hari) kepada klien lama Anda.

POST /api-key/extend
Requires x-master-key

Contoh Request Body

{
  "target_api_key": "KEY-XXXXX",
  "tambah_hari": 30
}
Respon Sukses (200 OK) 
{
  "status": true,
  "message": "Masa aktif berhasil diperpanjang."
}

Delete API Key

Menghapus klien dari database beserta seluruh koneksi WhatsApp (Sesi) yang tersangkut di sistem.

POST /api-key/delete
Requires x-master-key

Contoh Request Body

{
  "target_api_key": "KEY-XXXXX"
}

Kirim Pesan

Mengirim pesan teks biasa ke WhatsApp target.

POST /kirim-pesan
Requires x-api-key

Body Parameters (JSON)

FieldTypeStatusDeskripsi
nomorStringWajibNomor tujuan (Contoh: 628123456789).
pesanStringWajibIsi teks pesan WhatsApp. Format *Tebal*, _Miring_ didukung.

Contoh Request Body

{
  "nomor": "628123456789",
  "pesan": "Halo, ini pesan dari *API Gateway*! \n\nBaris baru didukung."
}
Respon Sukses (200 OK) 
{
  "status": true,
  "message": "Pesan dimasukkan ke antrean pengiriman.",
  "queue_id": 1542
}
Respon Error (402 Payment Required) 
{
  "status": false,
  "message": "Kuota pesan Anda habis. Silakan upgrade."
}

Kirim Massal (Broadcast)

Menembakkan ribuan pesan sekaligus ke berbagai nomor dengan menggunakan Antrean (Queue) agar anti-banned.

POST /kirim-massal
Requires x-api-key

Contoh Request Body

{
  "delay_ms": 3000,
  "data": [
    { "nomor": "6281xxx", "pesan": "Halo bos" },
    { "nomor": "6282xxx", "pesan": "Halo gan" }
  ]
}

Kirim Media

Mengirimkan gambar, dokumen PDF, atau video berdasarkan tautan (URL Publik).

POST /kirim-media
Requires x-api-key

Contoh Request Body

{
  "nomor": "628123456789",
  "media_url": "https://domain.com/gambar.jpg",
  "media_type": "image",
  "caption": "Silakan lihat brosur terlampir."
}

Kirim Polling

Membuat formulir voting langsung di dalam jendela chat pelanggan Anda.

POST /kirim-polling
Requires x-api-key

Contoh Request Body

{
  "nomor": "628123456789",
  "name": "Berapa umur Anda?",
  "values": ["18-25", "26-35"],
  "selectableCount": 1
}

Kirim ke Grup

Kirim pesan ke dalam Grup WhatsApp. Anda membutuhkan ID Grup untuk melakukan ini.

POST /kirim-grup
Requires x-api-key

Contoh Request Body

{
  "group_id": "123456789@g.us",
  "pesan": "Halo member grup!"
}

Add Device

Menginisiasi permintaan sesi WhatsApp ke server untuk mendapatkan Pairing Code.

POST /device/add
Requires x-api-key

Contoh Request Body

{
  "nomor_hp": "6285551234"
}
Respon Sukses (201 Created) 
{
  "status": true,
  "message": "Sesi berhasil diinisiasi.",
  "pairing_code": "A1B2-C3D4"
}

Delete Device

Keluar (Log-Out) perangkat dari server dan memutus koneksinya.

POST /device/delete
Requires x-api-key

Contoh Request Body

{
  "nomor_device": "6285551234"
}

Set Webhook

Hubungkan aplikasi pihak ketiga Anda ke server agar menerima setiap notifikasi pesan WhatsApp yang masuk secara Real-Time.

POST /webhook/set
Requires x-api-key

Contoh Request Body

{
  "webhook_url": "https://serveranda.com/api/wa-callback"
}

Add Auto-Reply (Chatbot)

Buat skenario Chatbot langsung di dalam Gateway (Respon cepat tanpa coding).

POST /auto-reply/add
Requires x-api-key

Contoh Request Body

{
  "nomor_device": "628123456789",
  "keyword": "harga",
  "response": "Harga paket kami mulai dari Rp 50.000,-",
  "match_type": "contains"
}
