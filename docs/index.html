API Documentation

Integrasikan sistem Anda dengan WhatsApp Gateway kelas Enterprise. Dilengkapi fitur Rotator, Spintax, dan Auto-Read.

Base URL

Gunakan URL di bawah ini sebagai awalan (prefix) untuk semua rute request API Anda.

https://api.krisnamarket.my.id

🔐 Otorisasi Global

Semua endpoint klien wajib menyertakan x-api-key di Header. Keamanan kami mengandalkan validasi ketat berlapis untuk menjamin perlindungan privasi data.

Header NameWajib?Deskripsi
x-api-key Ya Token API yang didapatkan klien saat berlangganan.
sender_id Opsional Bisa disertakan di Headers, Query URL, atau Body JSON. Jika dikosongkan, fitur ROTATOR akan otomatis aktif!

Real-Time WebSockets (Socket.io)

Server mendengarkan di port yang sama. Anda dapat menggunakan socket.io-client di Frontend untuk menangkap status perangkat secara real-time. Server akan melakukan Emit (memancarkan) event bernama device_status.

// Contoh di Frontend (React / Vanilla JS)
const socket = io("https://api.krisnamarket.my.id");

socket.on("device_status", (data) => {
    console.log(data);
    // { "device": "62812xxx", "status": "WAITING_PAIRING", "code": "A1B2-C3D4" }
    // { "device": "62812xxx", "status": "CONNECTED" }
    // { "device": "62812xxx", "status": "DISCONNECTED" }
});

/api-key/generate

Mencetak API Key baru untuk Pelanggan. Rute ini rahasia dan hanya bisa diakses oleh Master.

PENTING (ENTERPRISE SECURITY):

API Key yang dihasilkan akan di-hash (enkripsi) di database. `plain_key` HANYA akan ditampilkan 1 kali pada response ini. Simpan baik-baik!

Header / BodyWajibDeskripsi
x-master-keyYa (Header)Kunci rahasia dari file .env VPS.
paketOps. (Body)Free, Lite, Pro, Premium. Default: Free.
labelOps. (Body)Nama pemilik API Key (Default: User).

Request Example (JSON)

{
    "paket": "Lite",
    "label": "PT Krisna Dev"
}

Response Example

{
    "status": true,
    "message": "API Key Lite berhasil dibuat.",
    "plain_key": "KEY-1A2B3C4D5E",
    "data": { "key": "8f4a... (Hashed)", "label": "PT Krisna Dev", "paket": "Lite", "limit_pesan": 10000, "max_devices": 1 }
}

/webhook/set

Daftarkan URL Webhook Anda untuk menerima notifikasi secara real-time setiap kali ada pesan masuk atau perubahan status pesan/koneksi.

💡 TIPS SYNCHRONOUS REPLY:

Jika server Webhook Anda merespon HTTP POST dengan format JSON: {"reply": "Balasan otomatis Anda"}, maka bot akan membalas pesan tersebut secara instan tanpa perlu memanggil endpoint /kirim-pesan.

Header / BodyWajibDeskripsi
x-api-keyYa (Header)API Key Anda
webhook_urlYa (Body)URL server Anda (Gunakan null/kosong untuk mematikan)

Request Example (JSON)

{
    "webhook_url": "https://server-klien.com/webhook/wa"
}
POST

/auto-reply/add

Tambahkan kata kunci Auto-Reply. Mendukung pengiriman Teks, Gambar, Video, Dokumen, VCard, dan Lokasi.

Header / BodyWajibDeskripsi
x-api-keyYa (Header)API Key Anda
sender_idYa (Head/Body)Nomor WA pengirim Anda
keywordYa (Body)Kata kunci pemicu (contoh: PING)
responseYa (Body)Teks balasan atau *caption* media
match_typeOps. (Body)exact (Sama persis) atau contains (Mengandung kata). Default: exact
media_typeOps. (Body)Kosongkan untuk Teks. Isi dengan: image, video, document, location, vcard
media_urlOps. (Body)URL file gambar/video/dokumen, ATAU koordinat lat,long untuk lokasi, ATAU nomor_wa untuk vcard

Request Example (JSON - Media Image)

{
    "sender_id": "0812345678",
    "keyword": "PROMO",
    "response": "Berikut adalah katalog promo bulan ini!",
    "match_type": "exact",
    "media_type": "image",
    "media_url": "https://domain.com/katalog.jpg"
}

/queue/cancel

Berfungsi sebagai **Kill Switch** untuk membatalkan seluruh pesan yang berstatus `pending` di antrean. Sangat berguna jika Anda salah mengirim pesan massal dan ingin menyelamatkan limit kuota Anda.

Header / BodyWajibDeskripsi
x-api-keyYa (Header)Token API
deviceOps. (Body)Hanya membatalkan antrean dari nomor sender tertentu. (Kosongkan untuk hapus semua).

Request Example (JSON)

{
    "device": "0812345678" // Opsional
}

Response Example

{
    "status": true,
    "message": "Berhasil membatalkan dan menghapus 150 antrean pesan yang berstatus pending."
}
